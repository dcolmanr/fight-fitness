GUÍA RÁPIDA: CÓMO ENCENDER Y VER EL PROYECTO REACT (INTRANET)

Sigue estos pasos cada vez que quieras trabajar en tu código o mostrar
la página web en tu navegador.

PASO 1: ABRIR EL TERMINAL CORRECTO EN VS CODE

Abre tu Visual Studio Code con tu proyecto (fight-fitness-main).

En la lista de archivos a la izquierda, busca "package.json".

Haz clic DERECHO sobre "package.json".

Selecciona "Abrir en el terminal integrado" (Open in Integrated Terminal).
(Esto asegura que la consola se abra en la ruta exacta y no haya errores).

PASO 2: ENCENDER EL SERVIDOR LOCAL

En la consola que se acaba de abrir abajo, escribe el siguiente comando:

npm run dev

(si no funciona usa:
1- cd fight-fitness-mai
2- nnpm install
y le dices que si a todo, es para que se instalen las dependencias npm

Y SI SIGUE SIN FUNCIONAR VAS A POWERSHELL EJECUTADO COMO ADMINISTRADOR Y PONES LO SIGUIENTE:

Set-ExecutionPolicy RemoteSigned

eso ara que no te de restriciones al hacer el comando)

Presiona la tecla Enter.

PASO 3: ABRIR LA PÁGINA EN EL NAVEGADOR

En un par de segundos, la consola te mostrará un mensaje verde que
dice "VITE" y te dará un enlace local, normalmente es este:

➜  Local:   http://localhost:5173/

Pasa el cursor de tu mouse por encima de ese enlace.

Mantén presionada la tecla "Ctrl" (Control) en tu teclado.

Haz clic izquierdo sobre el enlace.

¡Tu navegador web (Chrome, Edge, etc.) se abrirá automáticamente
mostrando la página principal (index) de tu proyecto!

PASO 4: INGRESAR AL SISTEMA (CREDENCIALES)

Para probar la aplicación y entrar al Panel Principal, utiliza los
siguientes datos simulados en la pantalla de inicio de sesión:

Usuario: admin

Contraseña: 123

PASO 5: ¿CÓMO APAGAR EL SERVIDOR CUANDO TERMINES?

Mientras el servidor esté corriendo, no podrás escribir nuevos comandos
en esa consola. Cuando quieras terminar de trabajar:

Haz clic en la consola negra de VS Code.

Presiona "Ctrl + C" al mismo tiempo.

Si te pregunta "¿Desea terminar el trabajo por lotes? (S/N)",
escribe la letra "S" y presiona Enter.
